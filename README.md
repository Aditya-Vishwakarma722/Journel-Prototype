# Journal Prototype

A simple **Spring Boot** project created as part of my journey to learn **Java Backend Development**. This project serves as a hands-on learning environment where I implement Spring Boot concepts as I study them.

## About the Project

The Journal Prototype is a REST API-based application that allows basic journal entry operations. At the moment, data is stored using an in-memory `HashMap` as I focus on understanding Spring Boot fundamentals before integrating a database.

As I continue learning, this project will gradually evolve with better architecture, new features, and database integration.

## Current Features

- Create Journal Entries
- View All Journal Entries
- View Journal Entry by ID
- Delete Journal Entry by ID
- RESTful API Endpoints
- In-memory data storage using `HashMap`

## Tech Stack

- REST API
